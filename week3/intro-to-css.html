Intro to CSS

CSS Locations

CSS can be written 3 locations.

Each location has a weight to decide when a conflict happens what style should display.

  1. External File - CSS can be written in a file that is included in the HTML page. (LINK tag)
  2. Internal File - Written within the HEAD tag this block of code is set a side to write CSS on the page. (STYLE tag)
  3. Inline Style - An attribute is added to an element where the CSS can be applied to the element directly. (STYLE attribute)

CSS Selector

The way we connect the HTML and CSS is with a selector.

There are three basic type of selectors that we use.

  • Tag Name Selector - The HTML tag is used to connect the HTML and CSS. Most tags are valid to use. (h1, h2, p, etc)
  • Class Name Selector - An Attribute of class is applied to an HTML elements with a name. That name is used to connect the CSS. (class="className", .className{})
  • ID Name Selector - An attribute of id is applied to an HTML element with a name. That name is used to connect the CSS. (id="idName", #idName{})

Note: - All class name and id names are case sensitive.
