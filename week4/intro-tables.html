Intro to tables

Tables are a way to display a chart of data in HTML form.

Parts of the table

  • table - defines the start and stop of the table.
  • tr - table row - defines the start and stop of a row.
  • td - table data - defines the content area of the table. All content must be wrapped with the TD tag.
