Blocklevel Elements

All blocklevel elements will display on their own line.

Paragraph Tag

The paragraph tag is use to wrap most content on the page.

All content must be wrapped within some HTML tag. If nothing else is correct use the paragraph tag.

Break Tag

To add a line break within a paragraph you insert a break tag at the point where you want the line to end.

The break tag is an example of a tag that does not have a closing tag. It will need to close itself.

Example

Monday June 22, 1984
By Dan Hahn

Unordered List
