Working with Forms

HTML forms are used to collcet data and send it to a server to procces.

FORM tag

The FORM tag is a wrapper tag that is used to group the form elements together.

FORM Attributes

  • action - Location of page that data is being sent to.
  • method - How the data is sent to the ACTION page.
    • get - data is sent int he URL string
    • post - data is sent with the REQUEST. This data is not visiable to the user.
  • NAME or ID - used to identify one from from an other.

INPUT

The INPUT tag is an input type used with HTML forms. The INPUT tag can be used in may ways by setting the TYPE attribute.

All input are inline elements.

text

An input with a attribute type of "text" defines a standard one line input field.

A text input has a standard width of 20 characters but this can be changed with CSS.

Attributes

  • name* - used to define one element from an other. Also the way the page in the ACTION URL collects the data
  • maxlength - denfines the maximum number of charaters allowed to be entered
  • value - defines a "default" or set value of the element when the page loads.
  • id - define a unique name that is used with the LABEL tag.
  • disabled - makes the input not active. When an input is disabled the input is grayed out and can not be clicked in to.
  • readonly - disables the the user form being able to change the value in the input. The content can be copied by not changed

* = required
